Quick Installation Guide

Follow these step-by-step procedures to fully integrate the new dashboard elements.

1

Part 1: Backend Setup

Home Assistant Custom Integrations

  1. • Open your HACS Interface in the side navigation panel.
  2. •
    Click the 3 dots menu in the upper right, choose Custom Repositories, and add:
    https://github.com/DonTranQuiL/Grocy
  3. • Set the dropdown classification selector to **Integration** and click install.
  4. ⚠️ Restart your Home Assistant instance to load the updated Python structures.
2

Part 2: Lovelace Configuration

Dashboard JavaScript module deployment

  1. •
    Copy your custom `grocy-action-card.js` script to the HA config directory:
    /config/www/grocy-action-card.js
  2. • Navigate to Settings > Dashboards > Resources in HA.
  3. • Create a new Resource element, configure the locator field as `/local/grocy-action-card.js`, and label it as a **JavaScript Module**.
  4. • Flush browser storage cache (`Ctrl + Shift + R`) to update standard asset pointers.

Dashboard Integration Blueprints

Copy and deploy these production-ready configurations into your Home Assistant installation.

LOVELACE YAML 
type: custom:grocy-action-card
layout_options:
  grid_columns: 3
NFC SMART PANTRY 
alias: "Grocy: NFC Tap - Feed Dogs"
mode: single
trigger:
  - platform: tag
    tag_id: "YOUR_NFC_TAG_HEX"
action:
  - service: grocy.consume_product_from_stock
    data:
      product_id: 42 
      amount: 1
      transaction_type: CONSUME
  - service: grocy.execute_chore
    data:
      chore_id: 15
AI SCANNER SCRIPT 
alias: "Grocy: Process with AI"
icon: mdi:receipt-text-scan
mode: single
sequence:
  - action: google_generative_ai_conversation.generate_content
    data:
      prompt: >
        Read this grocery receipt. Extract the items purchased, their quantities, and the prices. 
        Return ONLY a raw, valid JSON object matching this exact schema:
        {"items": [{"name": "Item Name", "amount": 1.0, "price": "2.99"}]}
        Do not include markdown blocks, backticks, or conversational text. Return ONLY the JSON.
      image_entity_id: image.receipt_scanner
    response_variable: ai_response
  - action: grocy.add_products_by_name
    data: "{{ ai_response.text | from_json }}"
